# Steam Store Data Analysis (As of 05/2019)

This project is currently a work-in-progress.

This project aims to clean raw Steam store data and enforce consistency within columns in order to allow for analysis via queries;
answering questions such as:

- What is the average price of games on the Steam store?
- Is there a correlation between release date and popularity?
- Do certain publishers charge more than others on average?
